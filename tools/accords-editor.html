Charger data/sauces.json

Cet outil édite data/sauces.json, la source unique de vérité : il en lit les sauces (nodes), le vocabulaire des plats (tagLabel, tagGroups) et la table accords. Vous retravaillez la grille, puis l'outil exporte un sauces.json mis à jour (les nodes sont laissés intacts). Reconstruisez ensuite la page avec node tools/build.js.

Aucune donnée n'est dupliquée ici : la grille reflète toujours l'état réel du fichier chargé.

Glissez-déposez sauces.json ici, ou

Astuce : servez le dossier (python3 -m http.server) et le fichier se charge tout seul.

Cliquez une case pour basculer un accord. Les lignes regroupent les sauces par famille (couleur), les colonnes regroupent les plats par catégorie. Utilisez Gérer les plats & catégories pour ajouter, renommer, supprimer ou réordonner les colonnes. Les blocs régénérés sont triés de façon déterministe pour des diffs propres.
